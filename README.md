# Projeto-de-IA

# Algoritmo Genético: Maximizador ou Minimizador de Funções
# Objetivo

Desenvolver um algoritmo genético capaz de encontrar valores inteiros que maximizem ou minimizem uma função matemática dentro de um intervalo definido pelo usuário.
A representação dos indivíduos será baseada em cromossomos binários. Cada indivíduo corresponderá a um número inteiro e as operações genéticas (crossover e mutação) serão realizadas sobre sua representação binária.
O projeto deverá permitir a comparação entre diferentes estratégias de seleção, crossover e mutação, possibilitando posteriormente a realização de análises estatísticas para determinar quais combinações produzem melhores resultados.
